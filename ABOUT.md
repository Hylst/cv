# À Propos du Projet CV Interactif

Ce projet est le CV interactif de **Geoffroy Streit**, Ingénieur pluridisciplinaire en reconversion IT.
Il a été conçu pour démontrer des compétences en développement Web (HTML/CSS/JS), en design UI/UX, et en intégration de fonctionnalités modernes.

## Fonctionnalités Clés
*   **Design Responsive & Moderne** : Utilisation de CSS Grid/Flexbox, variables CSS, et Glassmorphism.
*   **Interactivité** : Animations au défilement, modales pour les détails, filtrage des projets.
*   **Thèmes** : Support des modes Clair, Sombre et un mode "Rétro" caché (Easter Egg).
*   **Architecture Modulaire** : Code JavaScript organisé en modules ES6 (`main.js`, `data.js`, `ui.js`, `utils.js`, `particles.js`).
*   **Données Séparées** : Tout le contenu (compétences, parcours, projets) est stocké dans `data.js` pour une maintenance aisée.

## Stack Technique
*   **Frontend** : HTML5, CSS3 (Vanilla), JavaScript (ES6+).
*   **Assets** : FontAwesome pour les icônes, Google Fonts (Inter, Roboto).
*   **Outils** : Aucun framework lourd (React/Vue) n'est utilisé pour garder le projet léger et performant, mais la structure est prête pour une migration si nécessaire.
